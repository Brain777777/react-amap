# ControlBar 组件

Map 组件是其他组件的基础，其他地图组件必须作为 Map 的子组件使用，Map 组件会给所有的子组件注入 `map` 属性，在拥有访问这个属性的能力后，就可以根据高德原生 API 做高德允许你做的一切事情。

## 何时使用

-  在需要显示地图的地方使用；
